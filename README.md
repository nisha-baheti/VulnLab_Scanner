# VulnLab_Scanner

A modular web vulnerability scanner with a controlled lab environment to demonstrate real-world attacks.

## Features
- XSS Detection
- SQL Injection Detection
- Directory Traversal Detection
- Security Header Analysis
- Open Port Scanning

## Project Structure
- scanner/ → detection scripts
- vulnerable_apps/ → intentionally vulnerable demo apps

## Tech Stack
- Python
- Basic Web Technologies (HTML, JS)
